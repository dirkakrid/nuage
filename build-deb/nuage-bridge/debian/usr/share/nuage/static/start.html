Welcome to Nuage!

Nuage is a port of Arduino Yúns bridge library to Raspberry Pi. This enables you to emulate a Yún with a Raspberry Pi and an ATmega328P or some specially designed shield/HAT for your Raspberry. To find out more about Nuage read the documentation.

Local resources:

  • The key-value-storage - see, set and delete variables stored in the bridge
  • The REST API - entry point to retrieve stored data as JSON snippets
  • The YunClient/YunServer - entry point to pass URL strings to the MCU
